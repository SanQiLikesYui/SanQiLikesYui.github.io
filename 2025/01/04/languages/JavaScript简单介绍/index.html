数据类型

  1. string (字符串)

JS的字符串支持使用单引号'和双引号"来定义

当我们想在字符串里直接放变量,使用”`“来定义字符串即可,类似于Python的f-string

JS的数字包含了整数和小数,其他语言中叫做int和float,这些在JS中统称为number

布尔值嘛,和其他C Like语言一样,如Java,使用true和false来表示

  1. object (对象)

JS的对象更像是键值对,通过键对应值,通过.运算符访问属性
